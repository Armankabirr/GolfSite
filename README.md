# GolfSite

A single-page golf facility landing page with animated sections, custom cursor effects, and smooth scrolling. Built with HTML, CSS, and vanilla JavaScript, plus GSAP for scroll-based animations.

## Features
- Full-bleed hero video background
- Custom cursor with hover states
- Auto-scrolling marquee text
- Animated section reveals and parallax-like motion
- Interactive cards and call-to-action blocks

## Tech Stack
- HTML5
- CSS3
- JavaScript (vanilla)
- GSAP + ScrollTrigger (CDN)
- Remix Icon (CDN)

## Project Structure
- index.html
- style.css
- script.js
- Assests/ (images and video used in the layout)
- images/ (additional images)

## Getting Started
1. Open index.html in your browser.
2. Scroll to see the GSAP animations and hover interactions.

## Notes
- The page uses external assets from the Sidcup Family Golf site and a couple of CDN libraries.
- For local hosting, use a simple static server (optional).
